## One Definition, Many Outputs

```json
{
  "name": "Button",
  "props": [
    { "name": "label", "type": "string", "required": true },
    { "name": "variant", "type": "enum", "values": ["primary", "secondary"] }
  ],
  "stateMachine": {
    "initial": "idle",
    "states": {
      "idle": { "on": { "CLICK": { "target": "pressed" } } },
      "pressed": { "on": { "RELEASE": { "target": "idle" } } }
    }
  }
}
```

Compiles to: React · Vue · Angular · Web Components · native apps · TypeScript types

<Aside type="tip">
**Not just for design systems.** Use Pencel for cross-platform apps, micro-frontends, shared component libraries, or any multi-framework team.
</Aside>
